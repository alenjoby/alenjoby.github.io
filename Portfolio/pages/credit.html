Daisy UI

DaisyUI is a CSS framework that is built upon Tailwind CSS. It offers ready-made, modifiable user interface components, saving time and effort during development.

Tailwind CSS

Tailwind CSS is a utility-first CSS framework that includes low-level utility classes for creating responsive layouts and UI components.

AOS

AOS is a JavaScript package that lets you apply animations to items as they scroll across a webpage.

Vanilla CSS

Vanilla CSS is the raw power of CSS, allowing you to style websites from scratch. It is lightweight and provides complete control, although it can be tedious for big tasks. Consider building blocks having limitless possibilities, but needing more work to develop complex structures.
